Abhishek Pradhan

I am a Software Engineer at the Microsoft Corporation on the Bing team. I was previously an undergraduate student at the University of Illinois - Urbana Champaign where I earned my Bachelor of Science in Computer Science.

  • Bing

  • Dogepack

    I worked on a dogecoin mining pool with Andy Hou and Kevin Chiu. I developed a Windows desktop miner to enable easy mining on our pool. I also contributed to the pool's initial setup and maintainence. Emma Zhang made our nice website icon here on the right.

  • Illinois Phone

    Illinois Phone was a mobile phone application developed with Joe Fernandez and Amey Chaugule. It's objective was to enable students to schedule adhoc meetings and control access to resources.

    I worked on aspects of the mobile phone application which was built on the Windows Phone platform along with a cloud component built on the Google App Engine. I also implemented a Windows desktop component which was responsible for managing windows user accounts to control resource usage.

    This project began as part of a course taught by Constantine Polychronopoulos and Yih-Chun Hu.

  • EcoG Cluster

    This project was part of a course that was led by William D. Gropp and Wen-mei W. Hwu. We built this cluster over the course of the semester with the objective of building a power efficient supercomputer.

    During the course I gained experience in CUDA and MPI programming and helped build the cluster which ranked as the third-greenest supercomputer at the time. The cluster's architecture was published in a journal and it was later used by researchers in high-energy physics.
